FAQ

Perguntas comuns

Preciso saber programar?

Não. Você vai abrir a pasta no Claude Code e seguir o passo a passo do setup.

Preciso ter Claude Code?

Sim. A Máquina roda dentro do Claude Code, então você precisa ter acesso ao Claude Code na sua conta.

Preciso pagar Fal.ai?

Só para gerar imagens com IA. Cada carrossel com imagens custa centavos. Sem Fal.ai, a Máquina também gera posts com foco em layout, texto e identidade visual.

Funciona sem Telegram?

Funciona. Os PNGs ficam salvos na pasta do projeto. O Telegram serve para receber tudo direto no celular.

Posso adaptar para minha marca?

Sim. O setup pergunta sobre sua marca, tom de voz, cores, nicho e identidade visual.

O pagamento é único?

Sim. Você paga uma vez e fica com o sistema para usar sempre.

Tenho garantia?

Sim. Você tem 7 dias para testar. Se não fizer sentido para você, pode pedir reembolso.
